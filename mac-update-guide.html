Updating BlinkBlink on Mac

Important: Please follow these steps in order to properly update BlinkBlink.

Step 1: Quit BlinkBlink

1. Click the BlinkBlink icon in your menu bar

2. Select "Quit" from the menu

3. Wait a few seconds for the app to close completely

Step 2: Drag and Drop

1. Drag BlinkBlink to your Applications folder

2. Click "Replace" if prompted

Step 3: (Optional) Remove Security Restrictions

If you receive a security warning when trying to open BlinkBlink, follow the steps below to resolve the issue:

1. Open Terminal (press Cmd+Space and type "Terminal")

2. Copy and paste this command:

xattr -c /Applications/BlinkBlink.app

3. Press Enter to run the command

Why is this needed? Apple has strict security measures for apps, which is great for safety but can be a challenge for smaller developers.
